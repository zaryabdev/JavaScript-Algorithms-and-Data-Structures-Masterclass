# tree sum

Write a function, `tree_sum`, that takes in the root of a binary tree that contains number values. The function should return the total sum of all values in the tree.

## test_00:

```python
a = Node(3)
b = Node(11)
c = Node(4)
d = Node(4)
e = Node(-2)
f = Node(1)

a.left = b
a.right = c
b.left = d
b.right = e
c.right = f

#       3
#    /    \
#   11     4
#  / \      \
# 4   -2     1

tree_sum(a) # -> 21
```
